Technologies
Gatsby.js, WordPress, WooCommerce, Firebase Storage

This project was developed for the company who needed to get orders from clients who need to create banners for their internet campaigns. The project was developed using Gatsby.js WordPress as a headless CMS with WooCommerce. Big files such as open files like logotypes, key visuals and other assets which frequently exceeding 1GB are stored on Firebase Storage.
